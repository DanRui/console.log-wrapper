All-in-one cross-browser console.log() wrapper demo

Open your console, then click one of the buttons below.

These will write the major JavaScript variable types to the console (string, object, function, etc). Depending on your console, each type will appear either as a plain strings (IE, mostly) or as interactive, explorable items (Firebug, some others). Note that any limitations are imposed by the console itself, not the log() function being demoed here.

You can see the log() function in the source code of this page. For a detailed explanation, see the blog post.
